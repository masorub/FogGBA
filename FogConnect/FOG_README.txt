FogConnect — FogGBA Wi-Fi ROM transfer
======================================

РУССКИЙ
-------
1. PSP и ПК в одной Wi-Fi сети (роутер).
2. На PSP: FogGBA → меню → Wi-Fi Receive (IP и порт 2121).
3. На ПК: run.bat (или python main.py).
4. Find / введите IP → Send ROM → выберите .gba или .zip.
5. Файл появится в roms/ на PSP.

ENGLISH
-------
1. PSP and PC on the same Wi-Fi network (router).
2. On PSP: FogGBA → menu → Wi-Fi Receive (IP and port 2121).
3. On PC: run.bat (or python main.py).
4. Find / enter IP → Send ROM → pick .gba or .zip.
5. File appears in roms/ on the PSP.

No CM File Manager needed.
